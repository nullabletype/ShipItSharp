Octopus Deploy command line tooling

Ship releases with less manual Octopus work.

ShipItSharp helps teams coordinate releases of multiple projects, deployments, project profiles, channel cleanup, and variable updates from a practical CLI built around Octopus Deploy.

Download latest release View source on GitHub

Key functionality

Purpose-built Octopus deployment helpers.

01

Create and promote releases

Create releases, deploy them to environments, or promote existing releases through your Octopus lifecycle.

02

Profile-based deployments

Deploy packages from profile files so repeatable multi-project deployments can be run from the terminal.

03

Project filtering

Filter commands by project group and channel where appropriate to keep large Octopus spaces manageable.

04

Release maintenance

Rename releases across projects, clean up old channels with no packages, and force release variable updates.

05

Automation friendly

Use interactive prompts when working manually, or non-interactive commands when wiring ShipItSharp into scripts.

06

Windows service mode

Run profiled deployments as a Windows service for scheduled or background deployment coordination.
