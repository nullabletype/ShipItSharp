ShipIt deploy

Deploy projects to environments

Usage: ShipIt deploy [command] [options]

Options:
  -?|-h|--help            Show help information.
  -a|--apikey             Octopus API key
  -u|--url                URL for your Octopus Deploy installation
  -n|--noprompt           Run without interactive prompts
  -c|--channel            The channel to deploy
  -e|--environment        Environment name
  -g|--groupfilter        Group filter keyword
  -s|--saveprofile        Don't deploy, but save a profile instead to use later
                          with the profile command
  -d|--fallbacktodefault  Fallback to the default channel if there are no
                          packages available for the selected channel for a
                          project. Be careful, as this disables validation of
                          packages for channels before deploy.
  -r|--releasename        Release version number (must follow semantic
                          versioning)
  -m|--machine            Optional deployment target machine Id or name
  -p|--prioritise         Move tasks to top of the queue

Commands:
  profile                 Deploy using a stored profile
  profiledirectory        Deploy using every auto profile in a directory
  specific                Deploy a specific release version to an environment

Run 'deploy [command] -?|-h|--help' for more information about a command.
